В данной версии приложения реализованы следующие функции: звонок, перевод звонка, повтор, список вызовов, всплывающие окна с информацией о пользователе, его фотографией(если она отсутствует, то используется фотография по умолчанию), возможность отклонять и отвечать на звонки с помощи этих же окон. Интеграция с используемым IP-телефоном(реагирование приложения на кнопки звонка, завершения звонка, установка на удержание). Программа поддерживает больше одного звонка с возможность переключения между ними благодаря своей многопоточности. Большая часть багов решена, но могут возникать проблемы при нажатиях на кнопки IP-телефона. Ниже прилагаются файл приложения и его конфигурационные файлы. Для корректной работы программы файлы конфигурации необходимо поместить в папку C:\temp, также нужно создать папку с фотографиями и , хотя бы, положить в неё фотографию по умолчанию. 
Была изменена структур xml-файла с информацией о клиентах.
